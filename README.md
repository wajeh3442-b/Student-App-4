Student Profile Flutter App

This Flutter application is a simple and attractive Student Profile App developed using Dart and Flutter framework. The app demonstrates navigation between screens, image handling, and dynamic UI updates using StatefulWidget.

Features
Beautiful Student Profile UI
Multi-screen navigation
Profile image display
Like button with counter
Responsive layout design
Modern card-style interface
Custom shadows and rounded corners

MaterialApp
Scaffold
StatelessWidget
StatefulWidget
Container
Row
Column
Text
ElevatedButton
Navigator
Image.asset
Functionality
Home Screen
Displays student image
Shows student name and role
Contains button to open detail screen
Detail Screen
Displays academic information
Shows like counter
Like button increases count dynamically
Back button returns to previous screen

Flutter
Dart
